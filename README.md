Weather App
Description: A real-time weather application that fetches data from a weather API (like OpenWeatherMap) to show current conditions based on user input.

Key Features: Search by city, dynamic background changes based on weather, and error handling for invalid cities.

Tech Used: HTML5, CSS3, JavaScript (Fetch API).



Random Quotes Generator
Description: A lightweight app that displays inspirational quotes at the click of a button.

Key Features: Integration with a Quote API, and smooth transitions between quotes.

Tech Used: JavaScript, DOM Manipulation, CSS Flexbox.
